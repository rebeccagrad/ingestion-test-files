# Configuration

Environment and runtime configuration reference.

## Environment variables

| Variable | Required | Description |
|---|---|---|
| `API_BASE_URL` | Yes | Base URL for the DXC API |
| `CMS_SITE_ID` | Yes | Target site UUID |
| `LOG_LEVEL` | No | Logging verbosity (default: info) |

## Site settings

Site-level configuration is managed via the CMS admin panel under **Settings → Site Configuration**.

## Verification

Check that all required environment variables are set before starting the service.
