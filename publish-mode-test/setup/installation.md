# Installation

Step-by-step instructions for installing the system.

## Prerequisites

- Node.js 18 or higher
- Access to the DXC CMS environment
- Valid API credentials

## Steps

1. Clone the repository
2. Run `npm install` in the project root
3. Copy `.env.example` to `.env` and populate required values
4. Run `npm run build` to compile assets

## Verification

After installation, navigate to `/health` — you should see a 200 response.
