# Quick Start

Get up and running in under five minutes.

## 1. Log in

Navigate to the CMS login page and enter your credentials.

## 2. Create your first content item

From the dashboard, select **New → Article** and fill in the required fields.

## 3. Publish

Click **Publish** in the toolbar. The content will be live immediately.
